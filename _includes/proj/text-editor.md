# Text Editor

Built: **February 2016**

A text editor built from scratch in Java that supports:

- Cursor display
- Text input & deletion
- Word wrapping
- Opening and saving a file
- Arrow key and mouse input movement
- Window resizing
- Vertical scrolling
- Undo and redo
